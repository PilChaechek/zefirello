## Подробные шаги
### 1. Установка Laravel 12 + Docker (Sail) + mysql + redis (кэширование)
- `curl -s "https://laravel.build/temp?with=mysql,redis" | bash`
- Перемещаем файлы проекта в корень и удаляем папку `temp`
- Меняем порты, если необходимо в `.env`

```bash
# Web Server (вместо 80)
APP_PORT=8001

# MySQL (вместо 3306)
FORWARD_DB_PORT=3307

# Redis (вместо 6379)
FORWARD_REDIS_PORT=6380

# Vite Frontend (вместо 5173)
VITE_PORT=5174
```
- Поднимаем контейнеры `sail up -d` и проверяем http://localhost:8001/

### 2. Git Initialization
- git init
- git add .
- Настраиваем .gitignore
- `git commit -m "Initial commit: Fresh Laravel install"`
- Связываем с удаленным репозиторием `git remote`

### 3. Настраиваем Redis

```ini
# В файле .env и env.example
SESSION_DRIVER=redis
CACHE_STORE=redis
QUEUE_CONNECTION=redis
```

### 5. Frontend init

```bash
# 1. Ставим Vue и плагин, vue-router
sail npm install vue@latest vue-router@latest @vitejs/plugin-vue
```

#### Настройка `vite.config.ts`
```js
import { defineConfig } from 'vite';
import laravel from 'laravel-vite-plugin';
import vue from '@vitejs/plugin-vue';
import tailwindcss from '@tailwindcss/vite';

export default defineConfig({
    plugins: [
        laravel({
            input: ['resources/css/app.css', 'resources/js/app.ts'],
            refresh: true,
        }),
        vue({
            template: {
                transformAssetUrls: {
                    base: null,
                    includeAbsolute: false,
                },
            },
        }),
        tailwindcss(), // Tailwind v4
    ],
    resolve: {
        alias: {
            '@': '/resources/js', // Удобный алиас для импортов
        },
    },
});

```

#### Настройка CSS (Tailwind v4)
- Редактируем файл: `resources/css/app.css`
```css
@import "tailwindcss";
```

#### Cоздаем Корневой компонент Vue
```vue
<script setup>
import { ref } from 'vue';

const count = ref(0);
</script>

<template>
    <div class="min-h-screen bg-zinc-900 text-zinc-100 flex items-center justify-center">
        <div class="text-center space-y-4">
            <h1 class="text-4xl font-bold text-emerald-500">
                Laravel <span class="text-zinc-100">+</span> Vue 3
            </h1>

            <button
                @click="count++"

            >
                Counter: {{ count }}
            </button>
        </div>
    </div>
</template>
```

#### Шаг 2: Переименование файлов (Миграция на TS)
- `vite.config.js` -> `vite.config.ts`
- `resources/js/app.js` -> `resources/js/app.ts`
- `resources/js/bootstrap.js` -> `resources/js/bootstrap.ts`

#### tsconfig.node.json - нужен для Vite
```json
{
    "compilerOptions": {
        "composite": true,
        "skipLibCheck": true,
        "module": "ESNext",
        "moduleResolution": "bundler",
        "allowSyntheticDefaultImports": true
    },
    "include": ["vite.config.ts"]
}

```

#### tsconfig.json
```json
{
    "compilerOptions": {
        "target": "ESNext",
        "useDefineForClassFields": true,
        "module": "ESNext",
        "moduleResolution": "Bundler",
        "strict": true,
        "jsx": "preserve",
        "resolveJsonModule": true,
        "isolatedModules": true,
        "esModuleInterop": true,
        "lib": ["ESNext", "DOM"],
        "skipLibCheck": true,
        "noEmit": true,
        "baseUrl": ".",
        "paths": {
            "@/*": ["resources/js/*"]
        }
    },
    "include": ["resources/js/**/*.ts", "resources/js/**/*.d.ts", "resources/js/**/*.vue"],
    "references": [{ "path": "./tsconfig.node.json" }]
}

```
