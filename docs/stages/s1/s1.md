## Этап 1: Развернуть среду разработки

1. Установка Laravel 12 + Docker (Sail)
- Ставим проект через curl 
- Поднимаем контейнеры через Sail
- Проверяем, что сайт открывается в браузере

2. Git Initialization
- git init
- git add .
- Настраиваем .gitignore
- git commit -m "Initial commit: Fresh Laravel install"
- Связываем с удаленным репозиторием

3. Настройка Laravel + очистка
- Удаляем Дефолтные миграции create_jobs_table, cache_table и т.д
- Настраиваем .env

4. Настройка Code Style (Pint & ESLint)
Настраиваем инструменты авто-форматирования.

5. Frontend Init
- Ставим Vue 3, Vite, Tailwind, shadcn-vue
- Проверяем сборку ассетов
- Настраиваем вход в приложение
